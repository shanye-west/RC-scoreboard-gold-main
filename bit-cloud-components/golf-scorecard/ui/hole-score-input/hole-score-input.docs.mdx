### Basic Two-Player Score Entry

This example demonstrates the `HoleScoreInput` component configured for a two-player game, showing how scores are entered and managed for a mid-round hole.

```tsx
<HoleScoreInput
  holeNumber={9}
  players={[
    { id: 'p1', name: 'Shanye West' },
    { id: 'p2', name: 'Taylor Made' },
  ]}
  scores={{
    'p1': 4,
    'p2': NaN,
  }}
  onScoreChange={(playerId, score) => console.log(`${playerId} score changed to ${score}`)}
  onPreviousHole={() => console.log('Previous hole clicked')}
  onNextHole={() => console.log('Next hole clicked')}
/>
```

### Finalizing Scores on the Last Hole
Shows the component being used on the 18th hole with existing scores, demonstrating the disabled 'Next' button as the round concludes.
```tsx
<HoleScoreInput
  holeNumber={18}
  players={[
    { id: 'p1', name: 'Shanye West' },
    { id: 'p2', name: 'Taylor Made' },
  ]}
  scores={{
    'p1': 3,
    'p2': 4,
  }}
  onScoreChange={(playerId, score) => console.log(`${playerId} score changed to ${score}`)}
  onPreviousHole={() => console.log('Previous hole clicked')}
  onNextHole={() => console.log('Next hole clicked')}
  minScore={1}
  maxScore={10}
/>
